O que é frontend?

    FrontEnd é toda a parte visual do site, que geralmente é feito usando html, css e javaScript. Essas três linguagens são responsáveis por fazer praticamente tudo que um usuário interage em um site.

O que cliente x servidor?

    Cliente-servidor é uma arquitetura de rede, onde um lado faz uma requisição e o outro devolve as informações.

O que HTML, CSS e JavaScript?

    - HTML: Respónsavel pela estrutura base do site.
    - CSS: Respónsavel pela estilisação do site.
    - JavaScript: É usada para criar a parte lógica e deixar o site mais interativo e dinâmico.

O que é W3C?

    É uma empresa que padroniza a confecçao de sites.
